# 细胞进化联机战场（PVE + PVP）

本版重点：

- 每台电脑只操控一个玩家
- 支持多人联机（同房间/同服务器）
- 新增组队系统（同队不会互伤）
- 地图：`4200 x 3000`
- 多资源（中文显示）：蛋白质 / 矿物质 / 气体
- 进化列表会动态变化（跟随你的进化路径）
- 进化项显示资源需求 + 具体作用说明
- 可持续进化到多种高阶生物：人类、豹、恐龙等
- 显示自己的血量/攻击/护甲
- 敌人追击频率和伤害已下调
- 新增存档（自动存档 + 手动存档）

## 安装运行

```bash
npm install
npm start
```

浏览器打开 `http://localhost:3000`。

## 连接方式

- 填昵称（默认 `hope`）
- 填房间（默认 `main`）
- 填队伍（默认 `alpha`）
- 点击“加入服务器”

同一服务器中：
- 房间名相同 => 同场游戏
- 房间名不同 => 独立房间
- 队伍名相同 => 组队（PVP 不互伤）

## 操作

- 移动：`WASD`（加速度 + 阻尼，移动更丝滑）
- 攻击：鼠标左键（近距离）或 `J`
- 进化：数字键 `1-0`
- 存档：左侧“立即存档”按钮（同时会自动定时和离线存档）

## 存档说明

- 服务端将存档写入项目根目录 `saves.json`
- 存档以“昵称”作为键；同昵称再次登录会自动读取
