# 二维细胞进化 PVE + PVP 游戏

一个纯前端（HTML/CSS/JavaScript）小游戏：

- 双人本地对战（PVP）
- 野外敌人战斗（PVE）
- 采集资源并触发两阶段进化
- 第一阶段可选器官方向：外骨骼 / 肺 / 毒腺
- 第二阶段可进化为多种生物形态（如巨钳蟹、疾走蜥蜴、突袭毒蛇等）

## 运行

直接用浏览器打开 `index.html` 即可。

## 操作

- 玩家1：`WASD` 移动，`F` 冲刺，`G` 攻击，`R` 进化
- 玩家2：方向键移动，`/` 冲刺，`.` 攻击，`Shift` 进化

## 进化规则

- 第一次进化：资源 ≥ 25，可选器官分支
- 第二次进化：资源 ≥ 60，基于第一阶段分支解锁最终生物
