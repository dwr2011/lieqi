# 细胞进化联机战场（PVE + PVP）

本版重点：

- 每台电脑只操控一个玩家
- 支持多人联机（同房间/同服务器）
- 地图：`4200 x 3000`
- 多资源：`protein / mineral / gas`
- 进化方案列表明确写出资源需求
- 进化后改变形态
- 敌人使用 `idle / wander / patrol / aggro` 状态，不会永远追玩家

## 安装运行

```bash
npm install
npm start
```

浏览器打开 `http://localhost:3000`。

## 连接方式

- 填昵称（默认 `hope`）
- 填房间（默认 `main`）
- 点击“加入服务器”

同一服务器中：
- 房间名相同 => 同场游戏
- 房间名不同 => 独立房间

## 操作

- 移动：`WASD`
- 攻击：`J`
- 进化：数字键 `1-0`
