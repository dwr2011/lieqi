# 细胞进化联机战场（PVE + PVP）

按你的反馈，这版做了以下核心改动：

- ✅ 每个电脑只操控一个玩家（单客户端仅一个角色）
- ✅ 支持多人联机（Socket.IO 实时同步）
- ✅ 地图扩大到 `4200 x 3000`
- ✅ 增加多资源体系：蛋白（protein）/矿物（mineral）/气体（gas）
- ✅ 增加进化方案数量，并在界面中明确写出每种进化资源需求
- ✅ 进化后会改变外观形态（如 crab/lizard/jelly/beetle/ray）
- ✅ 人机不再全程追杀：新增 `idle/wander/patrol/aggro` 状态，仅在范围内概率追击

## 安装运行

```bash
npm install
npm start
```

打开浏览器访问：

- 本机：`http://localhost:3000`
- 局域网：`http://你的电脑IP:3000`

> 想多人联机时，让其他人访问你的局域网 IP 即可。

## 操作说明

- 移动：`WASD`
- 攻击：`J`
- 尝试进化：数字键 `1-0`（对应左侧进化列表顺序）

## 进化说明

- 每个进化方案都需要 **三种资源**：protein/mineral/gas
- 资源不足时不会进化
- 进化会增强属性并改变形态
